üîπ What is Auto Commit?

Think of Auto Commit as saving a document automatically:

Auto Commit ON ‚Üí Every change you make is saved immediately.

Auto Commit OFF ‚Üí You must save manually, otherwise changes are not saved.


üîπ In MySQL Terms

AUTO COMMIT applies to DML (INSERT, UPDATE, DELETE) only.

By default, AUTO COMMIT = ON ‚Üí every DML query is saved automatically.

If you set autocommit = 0, then you need to use:

COMMIT ‚Üí to save changes

ROLLBACK ‚Üí to undo changes


‚úî Only DML can be rolled back
‚ùå DDL, DCL, DQL CANNOT be rolled back
